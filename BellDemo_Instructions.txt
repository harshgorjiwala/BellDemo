1. Change connection string in appsettings.json
update data source with your local/remote sql server name

2. run following command from your packate manager console to create database and entities.
add-migration migrationName
update-database

3. open ssms and connect to local/remote sql server mentioned in step 1
run following queries against database Bell to create serviceCategoryType and ServiceCategories.

Use Bell;
